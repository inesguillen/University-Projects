¿Quieres consultar un mes o el día actual?

⬅️🗓️ Inclina el móvil hacia la izquierda y di el mes para consultarlo

➡️📅 Inclina el móvil hacia la derecha para consultar el día actual

⬆️🔙 Inclina el móvil hacia atrás para volver al menú

➡️ Inclina el móvil hacia la derecha para ir al mes / día siguiente

⬅️ Inclina el móvil hacia la izquierda para ir al mes / día anterior

🔙 Inclina el móvil hacia atrás para volver al menú

👍 Levante el pulgar derecho para retroceder de mes

☝ Levante el indice para avanzar de mes
